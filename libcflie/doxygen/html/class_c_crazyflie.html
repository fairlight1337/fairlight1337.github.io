Detailed Description

Crazyflie Nano convenience controller class.

The class containing the mechanisms for starting sensor readings, ordering set point setting, selecting and running controllers and calculating information based on the current sensor readings.

bool CCrazyflie::isAtDesiredPosition ( )

Whether or not the copter has reached it's desired position.

Taking an threshold value into account (which by default is set to 0.05m), this function returns a boolean value showing whether the copter is in vicinity of the desired position set earlier. This value serves an informational purpose for client programs to see whether their control goal was reached yet.

Returns:
Boolean value denoting whether the control goal position was reached.

bool CCrazyflie::m_bControllerIgnoresYaw [private]

Whether or not the controllers ignore controlling the yaw of the copter. Since for navigation, it is not important in which direction the copter points, it might be switched off to give the controllers a larger degree of freedom. This setting is forwarded to the currently active (or thereafter activated) controller.
