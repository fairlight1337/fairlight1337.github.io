char * CCRTPPacket::sendableData ( )

Prepares a sendable block of data based on the CCRTPPacket details.

A block of data is prepared that contains the packet header (channel, port), the payload data and a finishing byte (0x27). This block is newly allocated and must be delete[]'d after usage.

Returns:
Pointer to a new char[] containing a sendable block of payload data
